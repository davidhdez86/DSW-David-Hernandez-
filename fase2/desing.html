Dar a conocer para que sirven las etiquetas en HTML5

HTML es un markup language, lo que significa que está escrito con códigos que puede leer una persona sin que sea necesario compilarlo primero.
En otras palabras, el texto en una página web está «marcado» con estos códigos para dar instrucciones al navegador web sobre cómo mostrar
el texto.Estas etiquetas de marcado son las propias etiquetas HTML.
